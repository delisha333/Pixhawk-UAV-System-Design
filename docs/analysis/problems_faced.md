# Problems Faced and Solutions

## Problem 1: Motors not responding equally
Some motors were spinning faster than others.

### Cause:
- ESC calibration mismatch

### Solution:
- Recalibrated all ESCs together

---

## Problem 2: Drone drifting during hover

### Cause:
- Improper sensor calibration

### Solution:
- Recalibrated accelerometer using KK controller interface

---

## Problem 3: Confusion in tuning

Initially, it was difficult to understand how tuning values affect flight.

### Solution:
- Made small incremental changes
- Observed behavior after each change

---

## Key Learning

Most issues were not hardware faults,  
but due to:
- Calibration errors
- Lack of tuning understanding

Breaking the system into smaller parts helped solve the problems.
